Smoke Away

My first "proper" game

This is a game I built why learning Unity

I have used this idea mainly to learn various aspects of Unity game engine.

Some of the features of this game: menu, win and loose screens, ability to pause, options to enable and disable sounds, power ups, score and lives.

On-screen controls are quite hard in web version because this game is designed for mobile devices

  • Controls:
  • Left mouse button: Navigate menu, on-screen buttons
  • Keyboard left arrow: move paddle to the left
  • Keyboard right arrow: move paddle to the right
  • Keyboard space: shoot ball/missile power up
